uint32_t  addr,
int  size 
)

Simple flash erase.

This function erases a section of a flash device. Note that erase operations are extended, on both sides, to align with the device's erase block boundaries.

This function is thread safe.

Parameters
[in]fiA handle to the flash device to erase
[in]addrThe address of the flash device to erase
[in]sizeThe number of bytes to erase
Returns
Returns FLASH_OK if successful, otherwise an error.

Simple flash program.

This function programs a section of a flash device. The program area does not have to align with flash erase boundaries. Areas already programmed will not be erased.

This function is thread safe.

Parameters
[in]fiA handle to the flash device to program
[in]addrThe address of the flash device to program
[in]bufThe buffer to the data to be programmed
[in]sizeThe number of bytes to program
Returns
Returns FLASH_OK if successful, otherwise an error.
