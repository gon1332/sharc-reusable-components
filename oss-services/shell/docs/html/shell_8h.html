◆ SHELL_CONTEXT

struct SHELL_CONTEXT

Shell context block.

This structure holds the shell context between calls to shell_poll(). The application should not set any of these members. The documentation for this structure is to aid in debugging.

Data Fields
int blocking

Blocking mode. This is set in shell_init()

char cmd[SHELL_MAX_LINE_LEN+1]
